Problem code ranges
===================
Each API has 200 numeric codes between 50000 and 60000.

| API                             | Code Range      |
|---------------------------------|-----------------|
| Content Management API          | `50000`-`50199` |
| Authentication API              | `50200`-`50399` |
| Bill Presentment API            | `50400`-`50599` |
| Payment Order API               | `50600`-`50799` |
